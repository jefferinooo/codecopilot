# CodeCopilot

An LLM-powered codebase assistant with hybrid retrieval, multi-mode reasoning
(Explain / Trace / Debug / Refactor), and an LLM-as-judge evaluation pipeline.

**Status:** scaffolding. See `docs/spec.md` for the full build spec.

## Stack
FastAPI · Next.js 14 · Postgres + pgvector · OpenAI embeddings · Anthropic/OpenAI for generation
